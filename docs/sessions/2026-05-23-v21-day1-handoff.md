## Architecture

Each subsystem in one line:

- **scene-graph/** — Gemini 2.5 Pro extracts room IDs, bearing vectors, visible portals, shot type, and room confidence per photo; consistency pass resolves ambiguous rooms on a second Gemini call.
- **candidates/** — rule-based generator produces typed pairs (same-room, walkthrough-via-portal, wide-to-detail, aerial-to-entry, exterior-walkaround); bearing-compat and portal-gate filters prune physically implausible pairings.
- **picker/** — pure-TS logistic regression + stump-boost model trained on operator labels; heuristic fallback (cold-start) until 10 labels collected; retrains every 10 new labels.
- **apprentice/** — Gemini 2.5 Pro few-shot labeler bootstrapped from operator labels; rolling agreement tracker drives mode-switcher (Director's Cut → Apprentice Review → Autopilot at 70%/90% agreement thresholds).
- **outcome-feedback/** — async DB-polling worker processes completed renders; Gemini outcome judge scores the clip and writes judge_score back; retrain hook triggers picker retrain when new judged outcomes arrive.
- **guardrail/** — per-clip sharp-based line-delta + frame-diff entropy checks; triggers multi-take reroll (max 2 retries) on geometric artifacts before delivering clip to downstream.
- **fall-through/** — routes any photo with room_confidence < 0.97, zero candidates, or picker score < 0.5 to V1 single-image pipeline (Kling 2.6 Pro default / Seedance 2.0 A/B via SKU selector).
- **telemetry/** — append-only audit log per label; rolling accuracy tracker (last 20/50/100 labels); nightly held-out eval; feature-importance snapshots from picker retrains.

## How Oliver starts using this

1. Apply migrations 067-072 to dev Supabase (or wait for staging promotion — but the tables are needed)
2. Set GEN2_V21_ENABLED=true in Vercel preview env
3. Navigate to /dashboard/development/lab → click V2 toggle → /dashboard/development/lab/v21
4. Pick a property from the dropdown → Extract scene graph (~30s, 1 Gemini call)
5. Label pairs in Director's Cut mode. Every 10 labels retrains the picker. Confirmed pairs auto-render via outcome-feedback worker.
6. After ~30 labels, switch to Apprentice Review mode (Gemini predicts what Oliver would label; Oliver agrees/disagrees in ~1s/click)
7. Observability panel shows rolling accuracy, feature weights, agreement rate, cold-start status
